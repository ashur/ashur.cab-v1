# ashur.cab

Built using [Markdown][markdown], [Handlebars][handlebars] and [Gulp][gulp].


## Requirements

Install Node and [npm][npm], then install Gulp:

```
$ npm install gulp
```

## Building

To build for local development:

```
$ gulp build:dev
```

To build for production:

```
$ gulp build:dist
```
